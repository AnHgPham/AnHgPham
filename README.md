## How I Work

- Start with a working baseline, then improve with measurable experiments.
- Keep demos reproducible with clear README notes, setup commands, and artifacts outside Git.
- Prefer practical AI systems over isolated notebooks: inputs, outputs, evaluation, and user flow matter.
- Build small, readable modules first; optimize only after the workflow is correct.
